# 米田引理 (Yoneda Lemma)

## 定義 (Definition)

米田引理是範疇論中的基本引理，由日本數學家米田信夫於1954年提出。

The Yoneda Lemma is a fundamental result in category theory, introduced by Nobuo Yoneda in 1954.

## 陳述 (Statement)

設 **C** 為局部小範疇，F: **C** → **Set** 為函子，A ∈ Ob(**C**)。則存在自然同構：

Let **C** be a locally small category, F: **C** → **Set** be a functor, and A ∈ Ob(**C**). Then there exists a natural isomorphism:

```
Nat(Hom(A, -), F) ≅ F(A)
```

其中：
- Nat(Hom(A, -), F) 表示從 Hom(A, -) 到 F 的自然變換的集合
- Hom(A, -): **C** → **Set** 是共變 Hom 函子
- F(A) 是函子 F 在對象 A 上的值

Where:
- Nat(Hom(A, -), F) denotes the set of natural transformations from Hom(A, -) to F
- Hom(A, -): **C** → **Set** is the covariant Hom functor
- F(A) is the value of functor F at object A

## 同構的構造 (Construction of the Isomorphism)

給定自然變換 α: Hom(A, -) → F，對應元素為 α_A(id_A) ∈ F(A)

Given a natural transformation α: Hom(A, -) → F, the corresponding element is α_A(id_A) ∈ F(A)

反之，給定 x ∈ F(A)，定義自然變換 α_x 如下：
對任意 B ∈ Ob(**C**) 和 f: A → B：

Conversely, given x ∈ F(A), define natural transformation α_x as follows:
For any B ∈ Ob(**C**) and f: A → B:

```
α_x,B: Hom(A, B) → F(B)
α_x,B(f) = F(f)(x)
```

## 充滿嵌入 (Fully Faithful Embedding)

米田嵌入 (Yoneda embedding) Y: **C** → [**C**^op, **Set**] 定義為：

The Yoneda embedding Y: **C** → [**C**^op, **Set**] is defined as:

```
Y(A) = Hom(-, A)
Y(f: A → B) = Hom(-, f): Hom(-, A) → Hom(-, B)
```

米田引理的推論表明此嵌入是充滿且忠實的 (fully faithful)。

A corollary of the Yoneda Lemma shows this embedding is fully faithful.

## 重要性 (Importance)

1. **表示性質**：米田引理說明函子完全由其在表示函子上的作用決定
   - **Representability**: The Yoneda Lemma shows that a functor is completely determined by its action on representable functors

2. **泛性質**：許多數學構造的泛性質可通過米田引理優雅地表述
   - **Universal Properties**: Many universal properties in mathematics can be elegantly formulated via the Yoneda Lemma

3. **嵌入定理**：任何局部小範疇都可以充滿忠實地嵌入到函子範疇中
   - **Embedding Theorem**: Any locally small category can be fully faithfully embedded into a functor category

## 應用 (Applications)

- 同倫論 (Homotopy theory)

## 直觀理解 (Intuitive Understanding)

米田引理告訴我們：要理解一個對象 A，只需要知道所有從 A 出發的態射。
這個想法類似於物理學中的操作主義：我們通過觀察一個系統如何與其他系統相互作用來理解它。

The Yoneda Lemma tells us: to understand an object A, we need only know all morphisms from A.
This idea is similar to operationalism in physics: we understand a system by observing how it interacts with other systems.

用數學的語言說：對象由其與其他對象的關係完全決定。

In mathematical terms: an object is completely determined by its relationships with all other objects.
